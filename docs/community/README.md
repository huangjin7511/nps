# 社区与项目

这一部分用于问题反馈、交流、贡献和项目支持。

如果你准备提问、反馈问题或参与贡献，建议先看完本页，再进入对应入口。

## 先按事情找

| 你要做什么 | 建议入口 |
| --- | --- |
| 报告 Bug、文档错误或提交功能建议 | [贡献](/community/contribute.md) |
| 快速交流部署经验或确认思路 | [交流](/community/discuss.md) |
| 直接提交文档或代码改进 | [贡献](/community/contribute.md) |
| 支持项目持续维护 | [捐助](/community/donate.md) |
| 了解贡献者和项目支持者 | [致谢](/community/thanks.md) |

## 建议的反馈顺序

1. 先查看 [FAQ](/reference/faq.md)
2. 再查看 [补充说明](/reference/notes.md)
3. 仍未解决时，到 [GitHub Issues](https://github.com/djylb/nps/issues) 提交问题

## 哪种入口更合适

- 需要可追踪的问题单、功能建议或缺陷反馈：优先使用 GitHub Issues
- 需要快速讨论、确认思路或交流经验：优先使用社区讨论渠道
- 想直接参与代码、文档或测试改进：先看 [贡献](/community/contribute.md)
