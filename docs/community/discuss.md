# 交流

NPS 的公开交流入口主要有两类：

- 即时讨论： [Telegram 群组](https://t.me/npsdev)
- 问题追踪： [GitHub Issues](https://github.com/djylb/nps/issues)

## 什么时候用哪一个

| 场景 | 建议入口 |
| --- | --- |
| 想快速交流使用经验或部署思路 | Telegram |
| 发现 Bug、文档错误或需要长期跟踪的问题 | GitHub Issues |
| 提交功能建议并希望后续可检索 | GitHub Issues |
| 不确定是不是 Bug，想先确认方向 | Telegram 或先看 [贡献](/community/contribute.md) |

## 提问前建议准备

- 当前版本
- 操作系统和部署方式
- 关键配置
- 复现步骤
- 报错日志

这样别人更容易直接判断问题。
